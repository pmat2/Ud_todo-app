# Spring todo application

## Requirements:
- Java 11

## Current modules:
- Spring Boot
- H2 database
- JPA
- Flyway
- Actuator

## How to run
Install jdk11. Either via IntelliJ or from web downloaded installer.

In project directory:
- Run *mvn clean package*
- Move to */target/* directory
- Run *java -jar <app_name>*
